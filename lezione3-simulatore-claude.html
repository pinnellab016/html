I Selettori CSS

Un selettore indica al browser quali elementi vuoi stilizzare.
Tre tipi fondamentali, ognuno con il suo scopo preciso.

<>
Selettore di Tag
Seleziona per tipo di elemento

Colpisce tutti gli elementi di quel tipo nella pagina.

/* tutti gli h1 diventano blu */ h1 { color: blue; font-size: 32px; }
Quando usarloStili globali: tutti i paragrafi o tutti i titoli uguali.
.
Selettore di Classe
Seleziona per attributo class=""

Colpisce tutti gli elementi con quella classe. Applicabile a quanti elementi vuoi.

/* inizia con il punto */ .evidenziato { background-color: yellow; }
Quando usarloStili ripetibili: un gruppo di elementi con lo stesso aspetto.
#
Selettore di ID
Seleziona per attributo id=""

Colpisce un solo elemento unico. L'ID non si ripete mai nella pagina.

/* inizia con il cancelletto */ #titolo-principale { color: green; }
Quando usarloStile unico: un elemento specifico che non si ripete.
