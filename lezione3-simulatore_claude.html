I Selettori CSS

Un selettore indica al browser quali elementi vuoi stilizzare.
Tre tipi fondamentali — ognuno con il suo scopo preciso.

<>
Selettore di Tag
Seleziona per tipo di elemento

Colpisce tutti gli elementi di quel tipo nella pagina. Scrivi h1 e ogni titolo h1 riceverà lo stile.

/* tutti gli h1 diventano blu */ h1 { color: blue; font-size: 32px; } /* tutti i paragrafi grigio scuro */ p { color: #333; }
Quando usarlo Stili globali — vuoi che tutti i paragrafi o tutti i titoli abbiano lo stesso aspetto.
.
Selettore di Classe
Seleziona per attributo class=""

Colpisce tutti gli elementi con quella classe. La stessa classe può essere applicata a quanti elementi vuoi, anche di tipo diverso.

/* inizia sempre con il punto */ .evidenziato { background-color: yellow; font-weight: bold; } /* HTML — applicata a più elementi */ <p class="evidenziato">Primo</p> <p class="evidenziato">Secondo</p>
Quando usarlo Stili ripetibili — un gruppo di elementi deve avere lo stesso aspetto ma non sono tutti dello stesso tipo.
#
Selettore di ID
Seleziona per attributo id=""

Colpisce un solo elemento unico. L'ID deve comparire una sola volta nella pagina — usarlo su due elementi è un errore.

/* inizia sempre con il cancelletto */ #titolo-principale { color: green; text-align: center; } /* HTML — usato su un solo elemento */ <h1 id="titolo-principale">Titolo</h1>
Quando usarlo Stile unico — vuoi stilizzare un elemento specifico che non si ripete, come l'intestazione principale.

Riepilogo — Quale selettore scegliere?
